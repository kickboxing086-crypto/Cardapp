# Cardapp - Sistema de Cardápio Digital Inteligente

O **Cardapp** é uma plataforma de alta performance desenvolvida para revolucionar o comércio local, permitindo que lojistas e empreendedores criem seus próprios cardápios digitais profissionais com gestão de pedidos integrada ao WhatsApp.

## 🚀 Funcionalidades Principais

- **Cardápio Digital Personalizado:** Identidade visual, cores e logo customizáveis.
- **Gestão de Pedidos em Tempo Real:** Receba notificações e gerencie o status da produção.
- **Integração com WhatsApp:** Pedidos formatados enviados diretamente para o mensageiro.
- **Pagamentos via PIX:** Geração de QR Code e integração para facilitar o checkout.
- **Painel Administrativo Robusto:** Controle de estoque, categorias, sabores e complementos.
- **Acesso Multilojas:** Painel de Supervisor Geral (CEO) para gestão de faturamento e clientes.
- **PWA Ready:** Instalável no celular para uma experiência de app nativo.

## 🛠️ Tecnologias Utilizadas

- **Frontend:** React 18, Vite, Tailwind CSS, Motion (animações).
- **Backend:** Node.js, Express.
- **Banco de Dados:** Firebase Firestore para persistência em tempo real.
- **Autenticação:** Firebase Auth.
- **Notificações:** Web Push API (VAPID).
- **Ícones:** Lucide React.

## 📦 Como Rodar o Projeto

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env` baseado no `.env.example` e adicione suas chaves do Firebase e Gemini.

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Build para produção:**
   ```bash
   npm run build
   ```
